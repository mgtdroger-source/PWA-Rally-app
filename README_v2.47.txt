STC Recorder PWA v2.47 — complete package

Changes in v2.47:
- Atomic Target time now uses the same six-digit completion behaviour as Rally Start Time.
- Six valid digits format automatically as HH:MM:SS and close the mobile keypad.
- The keyboard right arrow also completes the Target time entry and closes the keypad.
- Entering Target time does not perform the sync; Sync must still be touched at the exact master-clock time.
- Atomic Time Sync button operation is unchanged.
- Help wording now explains that Synced must first be held briefly until the button shows Sync.
- index.html, manifest.webmanifest and service-worker.js are aligned to v2.47.
