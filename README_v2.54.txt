STC Recorder v2.54

Change from v2.53:
- Added a central rally-day validity guard.
- Each daily package is active from 03:00 on its rally date until 02:59:59 the following day.
- Before activation or after expiry, Begin, STC recording, Review/Edit and Submit are blocked.
- Opening the next day-specific link remains seamless; each package retains its own identity and controls.
- The header clearly shows Not yet active or Expired.
- No GPS timing, audit, configuration schema or submission payload changes.
